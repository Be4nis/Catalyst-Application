Diagrams

Before This Course

Before starting this course I knew the how to draw the Bohr model and Lewis Dot diagrams. Naïvely, I thought that they were 100% accurate and the only way to draw elements. When I initially learned about the Quantum Model, I found it very hard since it contradicted everything I knew about modelling elements. So naturally, when I first saw the diagram I was confused and kind of mad beceause I thought there is no point in doing this if the Bohr model is accurate and easier. Overtime I figured out that elements aren't shaped exactly like the Bohr model and that the Quantum model was a better representation of elements.

The Quantum Model

The Quantum model has energy levels called orbitals just like the Bohr-Rutherford model. However, these orbitals can only contain a maximum of 2n² electrons. Each major shell in the model is divided into subshells/sublevels which can contain a maximum number of electrons. These subshells are filled according to the Aufbau principle. Although I have certainly not mastered the Quantum Model, I do understand the importance of it.

Orbital Box Diagrams

Orbital Box Diagrams are used to represent the electron configuration of an atom. I know from the quantum model that there are 4 different sublevel types (s, p, d and f), each capable of holding a different number of orbits. I learned that to draw an orbital box diagram I need to represent each sublevel with boxes and each electron with arrows.
